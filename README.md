# Homepage factory hero image

Place the final full-width factory image in this folder, for example:

`public/images/factory-hero.jpg`

Then configure:

`NEXT_PUBLIC_HOME_HERO_IMAGE=/images/factory-hero.jpg`

Recommended source image:

- Landscape orientation
- 2400 x 1400 px or larger
- JPG, WebP, or AVIF
- Main production subject positioned near the center or right side
- Keep important details away from the far edges for mobile cropping

The homepage automatically applies a dark readability overlay. If the variable is empty,
the existing dark fallback background remains active.
